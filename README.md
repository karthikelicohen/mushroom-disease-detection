 🍄 Mushroom Detection and Classification Project

 About This Project

This project is based on image classification of mushrooms using deep learning.

I have built this system in 3 stages:

1. Detect whether the image contains a mushroom or not  
2. Classify mushroom as edible or poisonous  
3. Identify the disease affecting the mushroom  

The main goal of this project is to apply CNN models for solving real-world agricultural problems.



 Technologies Used

- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy  
- Matplotlib  
- Jupyter Notebook  



Project Files

- 01_mushroom_detection.ipynb  
- 02_edible_vs_poisonous.ipynb  
- 03_disease_classification.ipynb  

---

Stage 1 – Mushroom Detection

In this stage, the model checks whether the input image contains a mushroom or not.  
Image preprocessing and CNN model training are performed.

---

 Stage 2 – Edible vs Poisonous

In this stage, the mushroom is classified into edible or poisonous category.  
The model is trained using labeled dataset and evaluated using accuracy and confusion matrix.

---

Stage 3 – Disease Classification

This stage identifies different types of mushroom diseases using multi-class classification.
